Si vous cherchez comment créer une application SaaS, commencez par le pack SaaS. Il vous fournit les éléments dont la plupart des produits d'abonnement ont besoin avant que le premier client ne s'inscrive : comptes, tables de base de données, facturation, courrier électronique, interface utilisateur et structure d'application typée.

## Générer le démarreur SaaS

Ouvrez le [pack SaaS dans Stack Builder](/new?preset=nextjs-saas&view=command) ou exécutez la commande ci-dessous.

```bash
bun create better-fullstack@latest my-saas-app \
  --ecosystem typescript \
  --frontend next \
  --backend self \
  --database postgres \
  --orm drizzle \
  --auth better-auth \
  --payments stripe \
  --email react-email \
  --api trpc \
  --css-framework tailwind \
  --ui-library shadcn-ui \
  --package-manager bun
```

## Ce que comprend le pack

- Next.js comme limite Web et serveur.
- PostgreSQL avec Drizzle pour les données produit relationnelles.
- Better Auth pour la connexion, les sessions et les flux de compte.
- Stripe pour la facturation des abonnements.
- React Email pour les modèles d'e-mails transactionnels.
- Tailwind CSS et shadcn/ui pour l'interface du produit.
- tRPC pour les appels tapés entre React UI et le code du serveur.

## Premières choses à personnaliser

1. Remplacez le nom du produit généré, les métadonnées et la navigation.
2. Ajoutez vos tables de produits principaux à côté du schéma Drizzle.
3. Configurez les secrets Better Auth et les origines approuvées.
4. Créez des produits Stripe et mappez-les à vos plans d'application.
5. Ajoutez des e-mails transactionnels pour l'inscription, la facturation et les invitations d'équipe.

## Quand c'est le bon pack

Utilisez ce pack pour les produits d'abonnement, les tableaux de bord B2B, les outils de développement payants, les espaces de travail d'équipe et les produits pour lesquels l'authentification et la facturation doivent faire partie de la première version de travail.

Pour une présentation pas à pas au niveau de la pile, lisez [Next.js avec Drizzle et Better Auth](/guides/typescript/nextjs-drizzle-better-auth/).
