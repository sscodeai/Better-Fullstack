当你想要一个从一开始就包含安全、persistence、migrations 和 tests 的 Spring 服务时，请使用 Java API pack。

<StarterPackCta id="java-api" />

## 生成 Java API starter

打开 [Stack Builder 中的 Java API pack](/new?preset=java-secure&view=command)，或运行：

```bash
bun create better-fullstack@latest my-java-api \
  --ecosystem java \
  --java-web-framework spring-boot \
  --java-build-tool gradle \
  --java-orm spring-data-jpa \
  --java-auth spring-security \
  --java-libraries spring-actuator flyway \
  --java-testing-libraries junit5 testcontainers
```

## Pack 包含什么

- Spring Boot 用于 API application。
- Spring Security 用于受保护 routes。
- Spring Data JPA 用于 persistence。
- Flyway 用于数据库 migrations。
- Gradle 用于 builds。
- JUnit 5 和 Testcontainers 用于 tests。

## 首先要自定义的内容

1. 用你的 aggregate roots 和 repositories 替换示例 entities。
2. 围绕真实 authentication model 配置 Spring Security。
3. 在依赖生成的 schema 行为之前，加入 Flyway migrations。
4. 让 Testcontainers 靠近 API contract tests。

## 什么时候适合这个 pack

选择它来构建 enterprise APIs、团队维护的 backend services、安全 admin systems，以及 Java/Spring 生态是预期生产基础的项目。

如需 stack 层面的 walkthrough，请阅读 [安全的 Spring Boot API](/guides/java/spring-security-api/)。
