Python プロジェクトは `uv` と Python ネイティブ テンプレートを使用します。 API フレームワーク、ORM、検証、AI ライブラリ、認証ヘルパー、タスク キュー、GraphQL、品質ツールの Python オプションを選択します。

## 前提条件

- パイソン。
- `uv`。
- リポジトリの初期化が必要な場合は Git を使用します。

## スクリプト化された例

```npm
npm create better-fullstack@latest my-python-api -- \
  --ecosystem python \
  --python-web-framework fastapi \
  --python-orm sqlalchemy \
  --python-validation pydantic \
  --python-ai none \
  --python-auth jwt \
  --python-task-queue none \
  --python-graphql none \
  --python-quality ruff \
  --python-testing pytest hypothesis \
  --python-caching redis \
  --python-realtime python-socketio \
  --python-observability opentelemetry \
  --python-cli typer rich \
  --ai-docs none \
  --version-channel stable \
  --no-install \
  --no-git
```

## 生成された動作

- `uv sync` は、インストールが有効な場合のインストール パスです。
- `pythonAi` は複数選択カテゴリであり、TypeScript `--ai` とは別のものです。
- 選択したオプションに空の条件付き Python テンプレート ファイルが必要ない場合、そのファイルはスキップされます。
- AI 命令ファイルは、`--ai-docs none` を含め、`--ai-docs` を通じて引き続き利用できます。
