服務選擇會圍繞核心應用新增 provider clients、環境變數、範例或整合式鷹架。

## AI

TypeScript AI 選擇包含 Vercel AI SDK、Mastra、VoltAgent、LangGraph、OpenAI Agents SDK、Google ADK、ModelFusion、LangChain、LlamStack Agents SDK、Google ADK、ModelFusion、LangChain、LlamStackIndexBPH_FS__ CLI 或 none。

Python 有單獨的多重選擇 AI 分類，包含 LangChain、LlamaIndex、OpenAI SDK、Anthropic SDK、LangGraph、CrewAI、Haystack 或 none。

## 產品服務

常見服務分區包括：

- Email：React Email、Resend、Nodemailer、Postmark、SendGrid、AWS SES、Mailgun、Plunk 或 none。
- Realtime：Socket.IO、PartyKit、Ably、Pusher、Liveblocks、Yjs 或 none。
- Jobs：BullMQ、Trigger.dev、Inngest、Temporal 或 none。
- CMS：Payload、Sanity、Strapi、TinaCMS、Directus 或 none。
- Analytics：Plausible、Umami 或 none。
- Feature flags：GrowthBook、PostHog、LaunchDarkly、Flagsmith、Unleash 或 none。
