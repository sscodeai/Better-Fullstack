UI 和體驗選擇會塑造產生的前端程式碼、設計 primitives 和客戶端應用模式。

## Styling 和 UI 庫

CSS 選擇包括 Tailwind、SCSS、Less、PostCSS-only 或 none。 UI 函式庫包括 shadcn/ui、shadcn-svelte、daisyUI、Radix UI、Headless UI、Park UI、Chakra UI、NextUI、Mantine、MUI、Ant Design、Base UI、Ark UI、React Aria 和 none。

shadcn/ui 還有額外 presets，用於基礎 primitives、視覺風格、圖示、顏色主題、中性色、字體和圓角。

## 表單、驗證和狀態

前端應用選擇包括：

- 狀態：Zustand、Jotai、Nanostores、Redux Toolkit、MobX、XState、Valtio、TanStack Store、Legend State 或 none。
- 表單：TanStack Form、React Hook Form、Formik、Final Form、Conform、Modular Forms 或 none。
- 驗證：Zod、Valibot、ArkType、TypeBox、Typia、Runtypes、Effect Schema 或 none。

## 體驗功能

動畫、i18n 和前端相鄰的 app-platform addons 在概念上屬於這裡，即使它們的精確 flags 出現在不同選項表中。

React Native 有自己的行動裝置 UI 選擇：Tamagui、Gluestack UI、Uniwind、Unistyles 或 none。
