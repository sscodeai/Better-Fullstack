React Native オプションは `--ecosystem react-native` で選択されます。これらは Expo アプリを生成しますが、TypeScript Web フロントエンド、バックエンド、データベース、または Web デプロイメント カテゴリは公開しません。

<Callout kind="info" title="Compatibility notes">
`native-uniwind` は `mobile-ui: uniwind` を選択します。 `native-unistyles` は `mobile-ui: unistyles` を選択します。 Tamagui と gluestack-ui は、スタイル システムの競合を避けるために `native-bare` を対象としています。
</Callout>
