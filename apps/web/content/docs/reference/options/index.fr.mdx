Cette référence répertorie les valeurs CLI acceptées par Better Fullstack. Utilisez ces valeurs dans les commandes, les URL Stack Builder et les charges utiles MCP.

Certaines étiquettes de générateur se normalisent avec la même valeur CLI, et toutes les options ne sont pas valides avec chaque frontend, backend, runtime ou écosystème. Exécutez une vérification de compatibilité avant d’échafauder des piles complexes.

<Callout kind="info" title="Compatibility matters">
Les tableaux répertorient les valeurs d'options valides. Le vérificateur de compatibilité décide toujours si une combinaison de pile complète est valide et peut rejeter ou ajuster les combinaisons avant la génération.
</Callout>
