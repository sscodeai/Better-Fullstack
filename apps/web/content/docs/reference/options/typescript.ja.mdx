TypeScript は、最も広範な Better Fullstack エコシステムです。これらの CLI 値は、コマンド、MCP ペイロード、および Stack Builder リンクで使用します。

<Callout kind="info" title="Compatibility notes">
この表には有効な値がリストされており、すべての有効な組み合わせがリストされているわけではありません。 tRPC は React 指向です。 `backend: self` はフルスタック フレームワーク用であり、`runtime: none` とペアになります。ワーカー ランタイム、UI ライブラリ、プロバイダー、アドオン、およびデプロイ ターゲットにはすべて、スタック固有の制約があります。
</Callout>

React Native および Expo オプションは、[React Native オプション](/docs/reference/options/react-native/) ページにあります。
