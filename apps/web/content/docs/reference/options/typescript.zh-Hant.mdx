TypeScript 是 Better Fullstack 涵蓋最廣的生態系。請在指令、MCP payloads 和 Stack Builder links 中使用這些 CLI 值。

<Callout kind="info" title="兼容性说明">
表格列出有效值，不代表每一種有效組合。 tRPC 面向 React。 `backend: self` 用於 fullstack frameworks，並與 `runtime: none` 搭配。 Workers runtime、UI libraries、providers、addons 和 deploy targets 都有 stack-specific 約束。
</Callout>

React Native 和 Expo 選項位於 [React Native 選項](/docs/reference/options/react-native/)頁。
