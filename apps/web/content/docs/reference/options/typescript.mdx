TypeScript is the broadest Better Fullstack ecosystem. Use these CLI values in commands, MCP payloads, and Stack Builder links.

<Callout kind="info" title="Compatibility notes">
The table lists valid values, not every valid combination. tRPC is React-oriented. `backend: self` is for fullstack frameworks and pairs with `runtime: none`. Workers runtime, UI libraries, providers, addons, and deploy targets all have stack-specific constraints.
</Callout>

React Native and Expo options live on the [React Native Options](/docs/reference/options/react-native/) page.
