TypeScript는 가장 광범위한 Better Fullstack 생태계입니다. 명령, MCP 페이로드 및 Stack Builder 링크에서 이러한 CLI 값을 사용하세요.

<Callout kind="info" title="Compatibility notes">
표에는 유효한 모든 조합이 아닌 유효한 값이 나열되어 있습니다. tRPC는 React 지향적입니다. `backend: self`은 풀스택 프레임워크용이며 `runtime: none`과 쌍을 이룹니다. 작업자 런타임, UI 라이브러리, 공급자, 추가 기능 및 배포 대상에는 모두 스택별 제약 조건이 있습니다.
</Callout>

React Native 및 Expo 옵션은 [React Native 옵션](/docs/reference/options/react-native/) 페이지에 있습니다.
