TypeScript ist das umfassendste Better Fullstack-Ökosystem. Verwenden Sie diese CLI-Werte in Befehlen, MCP-Nutzlasten und Stack Builder-Links.

<Callout kind="info" title="Compatibility notes">
Die Tabelle listet gültige Werte auf, nicht jede gültige Kombination. tRPC ist reaktionsorientiert. `backend: self` ist für Fullstack-Frameworks und Paare mit `runtime: none`. Worker-Laufzeit, UI-Bibliotheken, Anbieter, Add-ons und Bereitstellungsziele unterliegen alle stapelspezifischen Einschränkungen.
</Callout>

Die Optionen React Native und Expo sind live auf der Seite [React Native Optionen](/docs/reference/options/react-native/) verfügbar.
