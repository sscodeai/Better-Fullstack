---

Les options React Native sont sélectionnées avec `--ecosystem react-native`. Ils génèrent une application Expo et n'exposent pas les catégories TypeScript d'interface Web, de backend, de base de données ou de déploiement Web.

<Callout kind="info" title="Compatibility notes">
`native-uniwind` sélectionne `mobile-ui: uniwind`. `native-unistyles` sélectionne `mobile-ui: unistyles`. Tamagui et Gluestack-ui sont destinés à `native-bare` pour éviter les systèmes de style conflictuels.
</Callout>
