---
title: React Native 選項
description: Expo 和 React Native 中 frontend、navigation、UI、storage、testing、push、OTA 和 links 的選項值。
---

React Native 選項透過 `--ecosystem react-native` 選擇。它們會產生 Expo app，且不暴露 TypeScript Web frontend、backend、database 或 Web deployment 分類。

<Callout kind="info" title="兼容性说明">
`native-uniwind` 會選擇 `mobile-ui: uniwind`。 `native-unistyles` 會選擇 `mobile-ui: unistyles`。 Tamagui 和 gluestack-ui 面向 `native-bare`，以避免樣式系統衝突。
</Callout>
