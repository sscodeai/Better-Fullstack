This reference lists the CLI values accepted by Better Fullstack. Use these values in commands, Stack Builder URLs, and MCP payloads.

Some builder labels normalize to the same CLI value, and not every option is valid with every frontend, backend, runtime, or ecosystem. Run a compatibility check before scaffolding complex stacks.
