TypeScript est l'écosystème Better Fullstack le plus vaste. Utilisez ces valeurs CLI dans les commandes, les charges utiles MCP et les liens Stack Builder.

<Callout kind="info" title="Compatibility notes">
Le tableau répertorie les valeurs valides, pas toutes les combinaisons valides. tRPC est orienté React. `backend: self` est destiné aux frameworks fullstack et s'associe à `runtime: none`. Le runtime Workers, les bibliothèques d'interface utilisateur, les fournisseurs, les modules complémentaires et les cibles de déploiement ont tous des contraintes spécifiques à la pile.
</Callout>

Les options React Native et Expo sont disponibles sur la page [React Native Options](/docs/reference/options/react-native/).
