React Native 옵션은 `--ecosystem react-native`으로 선택됩니다. Expo 앱을 생성하고 TypeScript 웹 프런트엔드, 백엔드, 데이터베이스 또는 웹 배포 카테고리를 노출하지 않습니다.

<Callout kind="info" title="Compatibility notes">
`native-uniwind`은(는) `mobile-ui: uniwind`을(를) 선택합니다. `native-unistyles`는 `mobile-ui: unistyles`을 선택합니다. Tamagui와 Glustack-ui는 충돌하는 스타일 시스템을 피하기 위해 `native-bare`용으로 만들어졌습니다.
</Callout>
