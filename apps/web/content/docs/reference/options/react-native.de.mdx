React Native-Optionen werden mit `--ecosystem react-native` ausgewählt. Sie generieren eine Expo-App und machen keine TypeScript-Web-Frontend-, Backend-, Datenbank- oder Webbereitstellungskategorien verfügbar.

<Callout kind="info" title="Compatibility notes">
`native-uniwind` wählt `mobile-ui: uniwind` aus. `native-unistyles` wählt `mobile-ui: unistyles` aus. Tamagui und Gluestack-ui sind für `native-bare` gedacht, um widersprüchliche Stilsysteme zu vermeiden.
</Callout>
