這份參考列出 Better Fullstack 接受的 CLI 值。請在指令、Stack Builder URLs 和 MCP payloads 中使用這些值。

有些 builder labels 會規範化為相同的 CLI 值，並非每個選項都能與每個 frontend、backend、runtime 或 ecosystem 搭配。在 scaffold 複雜 stack 之前，請執行相容性檢查。

<Callout kind="info" title="兼容性很重要">
表格列出有效的選項值。 compatibility checker 仍會決定完整 stack 組合是否有效，並可能在生成前拒絕或調整組合。
</Callout>
