React Native options are selected with `--ecosystem react-native`. They generate an Expo app and do not expose TypeScript web frontend, backend, database, or web deployment categories.

<Callout kind="info" title="Compatibility notes">
`native-uniwind` selects `mobile-ui: uniwind`. `native-unistyles` selects `mobile-ui: unistyles`. Tamagui and gluestack-ui are intended for `native-bare` to avoid conflicting style systems.
</Callout>
