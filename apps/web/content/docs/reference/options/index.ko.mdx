이 참조에는 Better Fullstack에서 허용하는 CLI 값이 나열되어 있습니다. 명령, Stack Builder URL 및 MCP 페이로드에 이러한 값을 사용하세요.

일부 빌더 레이블은 동일한 CLI 값으로 정규화되지만 모든 옵션이 모든 프런트엔드, 백엔드, 런타임 또는 생태계에서 유효한 것은 아닙니다. 복잡한 스택을 스캐폴딩하기 전에 호환성 검사를 실행하십시오.
