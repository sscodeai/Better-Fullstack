Diese Referenz listet die CLI-Werte auf, die von Better Fullstack akzeptiert werden. Verwenden Sie diese Werte in Befehlen, Stack Builder-URLs und MCP-Nutzlasten.

Einige Builder-Bezeichnungen normalisieren sich auf denselben CLI-Wert, und nicht jede Option ist für jedes Frontend, Backend, jede Laufzeit oder jedes Ökosystem gültig. Führen Sie eine Kompatibilitätsprüfung durch, bevor Sie komplexe Stacks aufbauen.

<Callout kind="info" title="Compatibility matters">
